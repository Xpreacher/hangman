# Hangman Game

A classic word-guessing game you can play right in your terminal. Guess the hidden word one letter at a time before you run out of lives.

### Features
- **ASCII art**: Stick figure updates with each wrong guess  
- **Win/lose tracking**: Keeps score across rounds
- **No dependencies**: Just run and play

### How to Play
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/hangman.git
   cd hangman
   python hangman.py
